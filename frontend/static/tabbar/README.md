# TabBar 图标说明

## 图标文件

本目录包含微信小程序底部 TabBar 所需的图标文件：

- `detail.png` / `detail-active.png` - 明细页面图标（列表样式）
- `add.png` / `add-active.png` - 记账页面图标（加号样式）
- `mine.png` / `mine-active.png` - 我的页面图标（用户样式）

## 图标规格

- **尺寸**: 81x81px（微信小程序推荐尺寸）
- **格式**: PNG（支持透明背景）
- **格式**: PNG（支持透明背景）
- **颜色**:
  - 未选中状态: #7A7E83（灰色）
  - 选中状态: #3cc51f（绿色）

## 替换图标

当前图标为自动生成的占位图标。如需替换为更精美的图标：

1. **准备图标文件**
   - 尺寸：81x81px 或 162x162px（@2x）
   - 格式：PNG
   - 样式：单色图标（微信会自动处理颜色）

2. **替换文件**
   - 直接替换对应的 PNG 文件即可
   - 保持文件名不变

3. **图标设计建议**
   - 使用简洁的单色图标
   - 确保图标在小尺寸下清晰可见
   - 选中和未选中状态可以使用不同样式（如填充 vs 描边）
